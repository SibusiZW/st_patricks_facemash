# St Patrick’s FaceMash

A Django-powered web application inspired by early social ranking platforms, recreated for St Patrick’s High School, Fweru.

## 🧠 Overview
St Patrick’s FaceMash is a web-based comparison platform that allows users to vote between two profiles at a time. The system tracks votes and ranks profiles based on popularity.

## ⚙️ Tech Stack
- Django (Backend)
- HTML, CSS (Frontend)
- SQLite / PostgreSQL (Database)

## 🚀 Features
- 🗳️ Vote between two girls
- 🧾 Admin panel for managing profiles

## ⚠️ Note
This project is inspired by the original FaceMash concept. For ethical use:
- Use test or consented images only
- Avoid harmful or sensitive comparisons

## 📌 Use Case
Designed as a learning project to explore Django, database relationships, and user interaction systems.
